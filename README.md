# RugbyRef
Written in Java in Android studio, this rugby scorekeeping app allows a user to keep track of who is winning on the rugby pitch. 

## Scoring System

###Try = +5
###Conversion = +2 (Only after A Try)
###Pen. Kick = +3
###Drop-Goal = +2
